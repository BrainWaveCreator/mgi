Install Instructions

  

    
Gmail (web)

    
    
      
  1. Click Copy rendered above for the signature you want.
    2. 
      
  2. In Gmail, click the gear icon → See all settings.
    3. 
      
  3. Scroll to Signature → Create new, give it a name.
    4. 
      
  4. Click into the editor and paste (⌘V / Ctrl+V). The image and links come through.
    5. 
      
  5. Set the signature as default for new mail and replies. Save.
    6. 
    


    
Apple Mail

    
    
      
  1. In Apple Mail, go to Settings → Signatures.
    2. 
      
  2. Pick the account, click + to add a new signature.
    3. 
      
  3. Click Copy rendered above and paste into the signature editor.
    4. 
      
  4. Uncheck "Always match my default message font" at the bottom of the Signatures pane so the styles stick.
    5. 
    


    
Outlook (Web)

    
    
      
  1. Click the gear icon (top right) → View all Outlook settings.
    2. 
      
  2. Mail → Compose and reply → Signatures → + New signature.
    3. 
      
  3. Paste the rendered version. Set as default for new messages and replies.
    4. 
    


    
Outlook (Desktop)

    
    
      
  1. Outlook strips some inline CSS. Best path: paste the rendered version into Outlook → Preferences → Signatures.
    2. 
      
  2. If formatting drops, use the Copy HTML button and try again.
    3. 
    


    
Notes

    
    
      
  • The "Get a Quote" button is a styled link, not a real button — works in every email client.
    • 
      
  • Phone, email, website, and address are all clickable (tel:, mailto:, https://, and address opens in Maps).
    • 
      
  • If any image fails to load (email client blocks external images), the signature still reads well as text.
